# PREDICTING-CUSTOMER-BEHAVIOR-VIA-NEWS-EMOTION-AND-MISINFORMATION
An ML model that is used to detect Fake news through analyzing the Emotions of the news and then uses churn prediction to caluclate users influenced by the news

## REQUIREMENTS 
1. Pytorch
2. Transformers
3. fastapi
4. uvicorn
5. xgboost
6. pandas
7. numpy
8. scikit-learn
9. shap
